Laundry Services Website

A simple and responsive _Laundry Services landing page_ built using _HTML5_ and _CSS3_.  
This project showcases a clean navigation bar, a hero section with promotional content, and basic hover animations.

Features
1.Responsive navigation bar
2.Hero section with heading, description, and call-to-action button
3.Clean UI using Flexbox
4.Hover animation using CSS `transform`
5.Beginner-friendly and easy to customize

Technologies Used
1.HTML5
2.CSS3
3.Flexbox
4.CSS Transforms & Transitions

UI Overview

Navigation Bar

- Logo
- Home, Services, About Us, Contact Us links
- Username button

Hero Section

- Eye-catching heading
- Short description text
- “Book a service today!” button
- Laundry-related image

Button Hover Animation
CSS used for button animation:

.btn2{
transition: transform 0.4s ease, background-color 0.4s ease, border-radius 0.4s ease;
}

.btn2:hover{
transform: skew(15deg) rotate(-4deg) scale(1.1);
border-radius: 8px;
background-color: blueviolet;
}

This creates a smooth hover effect where the button skews, rotates, and scales slightly.

How to Run
To run this project, first download or clone the repository to your local system. After that, make sure all files (`index.html`, `style.css`, and `image.png`) are in the same folder. Simply double-click on the `index.html` file or open it with any modern web browser like Chrome, Edge, or Firefox. The laundry services website will open and run locally without needing any server or additional setup.
