Laundry Services Website

A clean and simple (Laundry Services landing page) built using HTML5 and CSS3.  
This project focuses on basic web layout concepts like navbar styling, inline-block, float positioning, and a hero section design without using Flexbox.

Features
1.Responsive-style layout using _inline-block & float_
2.Simple and clean _navigation bar_
3.Attractive _hero section_ with call-to-action
4.Beginner-friendly structure
5.No frameworks used (Pure HTML & CSS)

Technologies Used
1.HTML5
2.CSS3
3.Google-safe fonts
4.Basic layout techniques (float, inline-block)

Page Sections

Navigation Bar
1.Logo on the left
2.Menu links in the center
3.User button on the right
4.Built using `float` and `inline-block`

This project includes media queries for:
1.Desktop (default styling)
2.Tablet devices
3.Mobile devices

The layout automatically adjusts based on screen size.
1.Tablet View (768px – 1024px).
2.Mobile View(375px – 747px).

Learning Purpose

This project is ideal for beginners to understand:
1.Difference between **inline, inline-block, and block**
2.How `float` works in layouts
3.Basic navbar design without Flexbox
4.Structuring a landing page

How to Use
1.Create a project folder.
2.Add an index.html file for the page structure.
3.Add a style.css file for styling and media queries.
4.Link the CSS file with the HTML file.
5.Open the HTML file in a browser to view the page.
6.Resize the screen to see the responsive layout in action.
7.Customize colors, fonts, and sections as needed.
